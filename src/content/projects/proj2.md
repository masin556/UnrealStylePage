# P_Neon Strike

> [!NOTE]
> **Development Period**: 2024.04 - 2024.08 (5 Months)  
> **Development Team**: Solo Development  
> **Project Purpose**: High-speed cyber-parkour mechanics prototype focusing on predictive movement logic.

**Neon Strike** is a high-octane FPS.

## Overview
Speed is everything. The movement system is built on a custom character movement component that allows for wall-running, sliding, and air-dashing.

### Optimization
- Instanced Static Meshes for city generation.
- Niagra particle systems for neon effects.
